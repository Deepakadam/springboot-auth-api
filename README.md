# 🔐 Auth API - Spring Boot

A secure authentication system built using Spring Boot and MySQL with JWT-based authentication.

---

## 🚀 Features

- User Registration
- User Login
- JWT Token Generation
- Protected API (Authorization)
- Input Validation
- Global Exception Handling
- RESTful API Design

---

## 🛠️ Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- JWT (JSON Web Token)
